## Git Convention

|   Tag    |  Description   |
| :------: | :------------: |
|   feat   |      기능      |
|   fix    |      버그      |
| refactor |    리팩토링    |
| comment  |      주석      |
|   docs   |      문서      |
|   test   |  테스트 코드   |
|  rename  | 파일 혹은 폴더 |
|  build   |      배포      |
|  chore   |  사소한 수정   |

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
